Select Vacant Position by Division

First choose the Position Category / Division. Then click View Positions and click Select beside the position you want to apply for. The Position Title and Reference Number will be filled automatically from the official 266-position vacancy register.

Applicant Instructions

  • Closing Date: Submit your application before the advertised due date. Late applications may not be accepted.
  • Position Details: Select the division and vacant position in Step 1. The Position Title and Reference Number will auto-fill automatically.
  • Passport Size ID: Upload a clear recent passport-size photo in JPG or PNG format.
  • CV File Name: Name your CV as Surname_FirstName_CV.pdf. Example: Kuyama_Paul_CV.pdf.
  • Qualification File Name: Combine all certificates into one PDF and name it Surname_FirstName_Qualifications.pdf.
  • Training Certificates: Combine training or award certificates into one PDF where possible.
  • Document Format: Upload documents in PDF format unless otherwise stated.
  • Accuracy: Ensure all information is correct before submission. Only shortlisted applicants will be contacted.
