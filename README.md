# Department of Defence PNG Position Vacancies Website

GitHub-ready recruitment portal files for the Department of Defence Papua New Guinea.

## Files included

- `index.html` — homepage
- `vacancies_by_divisions.html` — position vacancies grouped by division
- `application-form.html` — online application form
- `success.html` — application submitted confirmation page
- `.nojekyll` — keeps GitHub Pages from applying Jekyll processing
- `logo.png` — Department logo/emblem image file
- `background.jpg` — background image file

## Upload to GitHub

1. Open your GitHub repository.
2. Upload all files in this folder to the root of the repository.
3. Commit the changes.
4. Go to **Settings > Pages**.
5. Under **Build and deployment**, select:
   - **Source:** Deploy from a branch
   - **Branch:** main
   - **Folder:** /root
6. Click **Save**.

## Live website link

For the `dodrecruitment/vacancies` repository, your website should be:

https://dodrecruitment.github.io/vacancies/

## Important note

The HTML layout has been kept in its original form. Replace `logo.png` and `background.jpg` with the official Department images if required, but keep the same filenames so the pages continue to load correctly.
